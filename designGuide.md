# ALPACA Mobile App Design Generation Prompt

## Project Overview

Design a complete mobile application UI/UX ecosystem for **ALPACA**, an agriculture, local product, and agro-tourism platform connecting UMKM owners, local farmers, agro-tourism operators, and consumers across Indonesia.

The final result must look like a real venture-backed startup product launching in 2026, not a generic AI-generated application.

---

## Design Philosophy

Create a premium, elegant, editorial-quality mobile experience inspired by:

- Apple Human Interface Guidelines
- Airbnb
- Linear
- Stripe
- Notion

The interface should communicate:

- Trust
- Professionalism
- Sustainability
- Simplicity
- Premium Digital Product Quality

Avoid:

- Generic AI-generated layouts
- Excessive gradients
- Glassmorphism
- Neumorphism
- Cartoon illustrations
- Oversized icons
- Visual clutter
- Heavy shadows
- Bright saturated color palettes

Focus on:

- Spacious layouts
- Strong typography hierarchy
- Refined whitespace
- Elegant photography
- Editorial composition
- Consistent design language
- Production-ready usability

---

## Color System

Primary:
Deep Natural Green (#2A5C45)

Secondary:
Warm Earth Amber (#C4813A)

Background:
Soft Off White (#F7F5F0)

Surface:
Pure White (#FFFFFF)

Text:
Near Black (#1C1917)

Success:
Natural Green

Warning:
Warm Amber

Error:
Muted Red

Rules:

- Maximum two brand colors per component
- Amber only for ratings, business metrics, financial indicators, and warnings
- Primary color reserved for navigation and main actions
- Avoid introducing additional accent colors

---

## Typography

Display Typography:
DM Serif Display

Interface Typography:
Plus Jakarta Sans

Hierarchy:

- Brand Wordmark: 52px
- Auth Titles: 28px
- Product Titles: 19px
- Section Headers: 15px
- Body Text: 14px
- Labels: 12px
- Micro Labels: 10px

Rules:

- Serif only for brand and authentication/onboarding headings
- Sans-serif for all functional UI
- Never mix serif and sans-serif within a single heading

---

## Layout Principles

- 8pt spacing system
- Generous whitespace
- Content-first hierarchy
- Soft corner radius between 16–24px
- Minimal elevation
- Clean borders
- Consistent visual rhythm
